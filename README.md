# BibliotecaControl
Interfaz de usuario moderna para la gestión de bibliotecas, desarrollada con Angular 17/18+. Implementa una arquitectura basada en Standalone Components, consumo de APIs REST con RxJS, tipado fuerte con TypeScript y un diseño responsivo enfocado en la experiencia del usuario (UX).

El código del cliente Angular vive en la carpeta **`web/`** (ver `web/README.md`).
